<h2>🚀 Phase 5A: Public Key Infrastructure (AD CS & NDES): </h2>
<h2>🔧 Key Implementations:</h2>

- <b>✔️Deployed a two-tier PKI hierarchy using an offline Standalone Root CA and online Enterprise Subordinate CA following enterprise security best practices.  </b>
- <b>✔️Generated, submitted, and approved subordinate CA certificate signing requests (CSR) to establish trust across the certificate chain </b>
- <b>✔️Created custom Client Authentication certificate templates with domain-wide auto-enrollment permissions </b>
- <b>✔️Configured Group Policy to distribute Root and Intermediate CA trust certificates to domain-joined systems </b>
- <b>✔️Enabled certificate auto-enrollment and automated renewal lifecycle management for domain clients </b>
- <b>✔️Verified successful client certificate enrollment through Enterprise CA issuance validation </b>
- <b>✔️Deployed and configured Network Device Enrollment Service (NDES) to support automated certificate enrollment for network infrastructure </b>
- <b>✔️Integrated certificate-based authentication for secure enterprise device trust </b>
